# BOQ Categorization Platform

AI-assisted BOQ categorization for electromechanical (MEP) contracting — Workshop staging, human approval, auditable publish to production.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

Next.js · React · TypeScript · Tailwind CSS · shadcn/ui · PostgreSQL (Supabase) · Drizzle · Supabase Auth · Zod · Vercel
